# Matplotlib_fundamentals
This notebook provides a beginner-friendly introduction to Matplotlib, one of the most widely used Python libraries for data visualization. It demonstrates how to install the library, import it, and create different types of plots with proper labels, legends, and grids.
# Contents Covered
- Adding:
  - Titles
  - Axis labels
  - Legends
  - Grids
- Displaying plots using plt.show()
# Objective
The goal of this notebook is to help beginners understand how to:
- Visualize data using Python
- Customize plots for better readability
- Build a strong foundation for data analysis and machine learning visualization tasks
